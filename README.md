# Veg & Fruit Market

A simple web app for selling vegetables and fruits.

## Features

- Browses fresh produce items
- Adds items to a shopping cart
- Updates quantities
- Displays subtotal and checkout button

## Run locally

1. Open `index.html` in your browser.
2. Or install the Live Server extension in VS Code and click "Go Live".

## Files

- `index.html` — structure and layout
- `styles.css` — responsive styling
- `script.js` — cart and product behavior
